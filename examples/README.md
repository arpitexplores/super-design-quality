# Examples

Use these prompts to test or demonstrate `super-design-quality`.

- `Use $super-design-quality to audit this page for accessibility.`
- `Use $super-design-quality to check mobile responsiveness.`
- `Use $super-design-quality to review this UI for visual consistency.`

## Evaluation Checklist

- The skill asks only for missing context.
- The output routes to the correct module.
- The answer is actionable and specific.
- Assumptions are stated clearly.
- Risks, validation steps, or next actions are included where relevant.
